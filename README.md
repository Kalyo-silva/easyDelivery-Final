# easyDelivery-Final

Trabalho desenvolvido como entrega final da matéria de Cloud Computing, ministrada pelo **Prof. Esp. Ademar Perfoll Junior**.

## Informações Gerais

Este trabalho tem como objetivo demonstrar a criação de uma API tematizada em torno de um pequeno serviço de entregas, denominado **Easy Delivery Co.**

O trabalho demonstra a criação da API, desenvolvimento e execução de testes unitários e a criação de uma *pipeline* de CI (Continuous Integration).

Em conjunto a este repositório, também foi criado um artigo documentando o processo de desenvolvimento e justificando as escolhas definidas durante a sua criação.

## Tecnologias Utilizadas

- **Node.JS** - Servindo como o backend da Aplicação
- **Express** - Para o dsenvolvimento da API
- **ESLint** - Para o processo de Linting do Código
- **Jest e Supertest** - Para a criação de testes unitários 
- **Github Actions** - Para a execução do Workflow de CI
- **Docker** - Para a conteinerização da aplicação.

## Instalação

**Pré Requisitos**
- Node.JS
- Docker
- Git

**Instruções**

1. Clone o repositório
```
git clone https://github.com/Kalyo-silva/easyDelivery-Final.git
```

2. entre na pasta api e realize a instalação das dependências
```
cd easyDelivery-Final/api/

npm install
```

3. Inicie a API
```
npm start
```
